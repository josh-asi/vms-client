## Setup

- Clone the repository by running the following command:

  `git clone https://github.com/josh-asi/vms-client`.

- Navigate to the directory created

  `cd vms-client`

- Install the node modules

  `npm install`

- Run the client

  `npm start`

## Technologies Used

- React
- TypeScript
